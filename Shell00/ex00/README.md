# Create file 'z'

> Target: Create files directly through Shell

<br>

The `cat` command allows you to create a file and already type the text it will contain.
```bash
 cat > z
```
Then just type the text and then press the `Ctrl+D` keys to end the typing.

```bash
Z
```
# 파일 만들고 내용 추가하기
```
echo "Z" > z
```
